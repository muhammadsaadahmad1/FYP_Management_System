File Guidelines

Supported File Types

  • Documents: PDF, DOC, DOCX, TXT
  • Presentations: PPT, PPTX
  • Archives: ZIP, RAR
  • Images: JPG, PNG, GIF (for diagrams)
  • Code: ZIP archives with source code

File Size Limits

  • Maximum file size: 50MB per file
  • Total storage: 500MB per group
  • Recommended report size: Under 10MB
  • Large files should be compressed

Security & Privacy

  • All files are encrypted during transmission
  • Only group members can access shared files
  • Supervisors have access to all reports
  • Files are automatically backed up
